Overview

The RedCu Meter is a device designed for measuring Redstone signal strength, primarily for debugging purposes.

Crafting

You can craft it using the RedCu Crafter, found in the Indicator block section.

IngredientsRedCu Crafter Recipe
1 Smooth Stone Plate
1 RedCu Ingot

Functionality

The RedCu Meter is particularly useful when working with Redstone CG gates, as it measures their direct outputs. For other blocks, it measures the indirect Redstone Signal, which is the signal from the block itself combined with signals from neighboring blocks. The displayed value represents the highest Redstone Signal available.

However, there are limitations:

  • It does not display correct values when pointed at a Redstone Torch (this may be fixed in future updates).
  • It cannot detect if a block is powered by quasi-connectivity unless the block itself determines it.

Extended Range for RedCu Wires

RedCu Wires operate on an extended Redstone signal range of 0-255, compared to the standard range of 0-15. When used to measure RedCu Wires, the RedCu Meter will display a decimal point value. The whole part of the displayed value represents the Redstone signal output to standard Redstone components.

For example:

If the RedCu Meter shows a value of 10.625 while pointing at a RedCu Wire, the connected Redstone Dust will show an output of 10.0.
