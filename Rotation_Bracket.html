Overview

The Rotation Bracket is a tool used to rotate blocks added by the Redstone CG mod. By default, right-clicking a compatible block with the Rotation Bracket in your main hand will rotate the block clockwise.

If you crouch while right-clicking, the block will instead rotate counterclockwise. This enables precise orientation control during circuit design.

Note: Only blocks from the Redstone CG mod can be rotated using the Rotation Bracket. The following blocks are exceptions and cannot be rotated:

  • RedCu Wire Transition
  • RedCu Crafter
