Server Start Crash

Severity:

Status:

Code:

Affected Versions:

Description: Creating or starting a server with Redstone CG caused an immediate crash before the world could be created, with logs showing:
Redstone CG (redstonecg) encountered an error during the common_setup event phase
java.lang.BootstrapMethodError: java.lang.RuntimeException: Attempted to load class net/minecraft/client/multiplayer/ClientLevel for invalid dist DEDICATED_SERVER

Cause: Client-side logic attempted to access a Level instance in a server-side environment within MessengerBlockEntityPigeon.handleData.

Consequences: Server crashes during startup and world creation fails.

Resolved: Fixed in version 0.3.1p1 (0.3.1 Patch 1). Client-side logic is now properly separated and executed only on the client, preventing server-side crashes.

The report of this issue has been provided by Mysticpasta1
