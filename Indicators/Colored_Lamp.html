Overview

Colored Lamp is an indicator block that increases its brightness based on the incoming Redstone signal strength.

The lamp can be recolored using Stained Glass Panes, allowing players to customize the appearance of indicators for decorative builds or complex circuits.

Configuration

By default, the Colored Lamp uses the color #F2BD74, which closely resembles the color of a standard Redstone Lamp.

Right-clicking the block with any Stained Glass Pane consumes the pane and applies its color to the lamp.

  • Applying a new pane replaces the previous one and drops the previously used pane.
  • Breaking the Colored Lamp will also drop the currently applied pane.

Yes, applying a Black Stained Glass Pane will make the lamp appear pitch black, and it will still emit light normally.

Internally, the lamp color is stored as a 3-byte array in the block entity (color). This value can be modified using commands to obtain any RGB color, not just those available from stained glass panes.

/data modify block <x> <y> <z> color set value [B;<R>B,<G>B,<B>B]
