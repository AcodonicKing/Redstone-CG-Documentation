Overview

Colorful Lamp is an indicator block that changes its color based on supplied Redstone signal.

The signal can be mapped to Color Code and equivalent Stained Glass Pane used on Colored Lamp:

Signal StrengthEquivalent Stained Glass PaneColor Code
0White Stained Glass Pane#FFFFFF
1Orange Stained Glass Pane#FF681F
2Magenta Stained Glass Pane#FF00FF
3Light Blue Stained Glass Pane#9AC0CD
4Yellow Stained Glass Pane#FFFF00
5Lime Stained Glass Pane#BFFF00
6Pink Stained Glass Pane#FF69B4
7Gray Stained Glass Pane#808080
8Light Gray Stained Glass Pane#D3D3D3
9Cyan Stained Glass Pane#00FFFF
10Purple Stained Glass Pane#A020F0
11Blue Stained Glass Pane#0000FF
12Brown Stained Glass Pane#8B4513
13Green Stained Glass Pane#00FF00
14Red Stained Glass Pane#FF0000

On Redstone signal 15, equivalent stained glass pane is White Stained Glass Pane (#FFFFFF), but the equivalent Redstone signal applied to Colored Lamp is 0. It also does not emit light when Redstone signal 15 is applied.
